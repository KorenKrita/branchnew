---
description: Fork the current OMP session into a new terminal split pane (to the right)
argument-hint: [name]
allowed-tools: Bash(branchnew *)
---
Run the shell command `branchnew $ARGUMENTS` via the bash tool to fork the current session into a new split pane on the right. The fork's name appears in the command output. In ONE short line, tell the user the fork is open. Do not do anything else.
